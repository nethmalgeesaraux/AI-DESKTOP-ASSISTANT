# Cerebro GUI Application

A modern desktop GUI application built with Python using Tkinter.  
This project focuses on creating an interactive user interface with image integration, message handling, and a clean modern design.

---

## Features

- Modern GUI design
- Message handling system
- Image integration
- Custom styling and layout
- Responsive widget positioning

---

## Technologies Used

- Python 3
- Tkinter
- Pillow (PIL)

---

## Project Structure

```bash
Cerebro/
│── image/
│   ├── logo.png
│   └── background.png
│
│── GUI.py
│── README.md
```

---

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Cerebro.git
```

Move to project folder:

```bash
cd Cerebro
```

Install dependencies:

```bash
pip install pillow
```

---

## Run the Project

```bash
python GUI.py
```
